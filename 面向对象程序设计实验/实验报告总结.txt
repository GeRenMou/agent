任务 3：重载函数与带默认形参值函数的二义性
================================================================================

【实验要求】
自举一例说明重载函数与带默认形参值函数在调用时可能产生的二义性问题，并

定义两个名为 print 的函数：
  - void print(int x)：重载函数
  - void print(int x, int y = 10)：带默认参数的函数
当调用 print(5) 时，编译器会遇到二义性问题。

【代码实现】
见 实验3/main.cpp

【实验总结 —— 避免二义性问题的注意事项】

1. 二义性产生原因：
   当重载函数中的某一个带有默认参数时，编译器可能无法确定调用哪个版本。
   例如：
     void print(int x);
     void print(int x, int y = 10);
   调用 print(5) 时两个函数均可匹配，编译器无法抉择，产生编译错误。

2. 避免二义性的注意事项：
   (1) 避免同时定义"参数个数不同但可通过默认参数补足"的重载函数。
   (2) 如果必须同时定义，调用时明确传参以消除歧义，如 print(5, 20) 或
       通过函数指针/类型转换明确指定。
   (3) 设计 API 时优先选择一种方式：要么用重载，要么用默认参数，不要混用。
   (4) 遵循"最少惊讶原则"：让调用者一眼就能判断会调用哪个版本。

【实验总结 —— 能否将 Point 类的定义放在 Rectangle 类后面？】

回答：不可以直接将 Point 类的完整定义放在 Rectangle 类后面。

原因：
  Rectangle 类的成员变量类型为 Point（bottomLeft 和 topRight），编译器
  在编译 Rectangle 类时需要知道 Point 类的完整定义（包括大小、成员等）。
  如果 Point 类定义在 Rectangle 之后，编译器在处理 Rectangle 时将无法
  确定 Point 类型，导致编译错误。

解决方案（如果可以）：
  使用前向声明（forward declaration）配合指针或引用：
    class Point;  // 前向声明
    class Rectangle {
        Point* bottomLeft;  // 使用指针而非对象
        Point* topRight;
        ...
    };
  Rectangle 中只使用 Point 的指针或引用时，前向声明即可满足编译要求，
  Point 的完整定义可以放在 Rectangle 后面。但使用对象本身（非指针/引用）
  时必须先有完整定义。

类的聚合设计的好处：
  1) 代码复用：避免重复定义坐标成员变量
  2) 结构清晰：矩形由两个点定义，语义直观
  3) 易于扩展：修改 Point 类不影响 Rectangle 类的接口

【实验总结 —— 友元函数声明的目的】

友元函数声明为类的"朋友"，使其可以访问该类的私有（private）和保护
（protected）成员。在本实验中，total 函数需要同时访问 CBank、IBank
和 ABank 三个类的私有成员 balance，因此需在每个类中声明为 friend。

使用场景：
  - 多个类之间需要共享私有数据时
  - 运算符重载（如重载 << 输出运算符）
  - 某些需要高效访问内部数据的全局函数

注意：友元破坏了封装性，应谨慎使用，仅在确实必要时声明。

【实验总结 —— 以常引用作为形参的目的】

const 引用的优点：
  1) 避免拷贝：引用传递不产生临时对象，节省内存和时间（尤其对大对象）
  2) 保护实参：const 修饰保证函数内部不会意外修改实参的值
  3) 可以接受临时对象：const 引用可以绑定到右值（如字面量、表达式结果）

示例：double total(const CBank& cBank, const IBank& iBank, const ABank& aBank)
  - const 保证函数不会修改任何银行对象的 balance
  - & 引用避免拷贝整个银行对象

静态数据成员属于类本身，不属于某个具体对象，被该类的所有对象共享。
使用目的：
  1) 共享数据：所有对象共同维护一份数据（如本实验的 totalSum 和

  1) 操作静态数据成员：如 getAveragePrice() 计算 totalSum/totalSales
  2) 无需实例化：可以通过 Product::getAveragePrice() 直接调用
  3) 工具函数：提供与类相关但不依赖对象状态的功能
  4) 工厂方法：用于创建和管理对象实例

【实验总结 —— 指针的分类和各类指针的用法】

1. 基本数据类型指针：
   int* p = new int(5);  // 动态分配
   delete p;             // 释放内存

2. 对象指针：
   Point* p = new Point(0, 0);
   p->x = 10;            // 通过指针访问成员
   delete p;

3. 指向指针的指针（多级指针）：
   int** pp;             // 通常用于二维数组或函数参数

4. const 指针与指向 const 的指针：
   const int* p1;        // 指向的值不可改
   int* const p2;        // 指针本身不可改
   const int* const p3;  // 两者都不可改

5. void 指针（void*）：
   可指向任意类型，使用时需强制转换

6. 函数指针：
   void (*funcPtr)(int); // 指向函数的指针，用于回调和多态

7. this 指针：
   成员函数中指向调用对象的指针

8. 智能指针（C++11）：
   unique_ptr: 独占所有权
   shared_ptr: 共享所有权，引用计数
   weak_ptr: 不增加引用计数的观察指针

【实验总结 —— 析构函数在本练习中的作用】

本实验中，Circle 类使用 float* radius 动态分配内存（new float），
析构函数负责释放该内存（delete radius），防止内存泄漏。

具体作用：
  1) 释放动态内存：构造函数中 new 分配的内存由析构函数 delete 回收
  2) 自动调用：对象生命周期结束时自动调用，无需手动干预
  3) RAII 原则：资源获取即初始化，将资源生命周期绑定到对象生命周期
  4) 防止内存泄漏：如果没有析构函数，每次创建 Circle 对象就会泄漏
     sizeof(float) 字节的内存

  注意：本例中若涉及拷贝构造和赋值运算符，还需遵循"三法则"
  （Rule of Three），同时实现拷贝构造函数和赋值运算符重载，
  避免浅拷贝导致的重复释放问题。

#include <iostream>
using namespace std;

// 求 n 的所有真因子之和（真因子：小于 n 的正整数中能整除 n 的数，
// 不包括 1 和 n 本身）
int sumOfFactors(int n) {
    int sum = 0;
    for (int i = 2; i <= n / 2; i++) {
        if (n % i == 0) {
            sum += i;
        }
    }
    return sum;
}

int main() {
    int n;
    cout << "请输入一个整数 n (1 < n <= 1000): ";
    cin >> n;
    if (n <= 1 || n > 1000) {
        cout << "输入不合法！n 应在 (1, 1000] 范围内。" << endl;
        return 1;
    }
    int result = sumOfFactors(n);
    cout << n << " 的真因子之和为: " << result << endl;
    return 0;
}

实验 4：Factorial 阶乘类（实验4/main.cpp）
------------------------------------------------------------------------------
#include <iostream>
using namespace std;

class Factorial {
private:
    int value;
    long long fact;

    void calculateFactorial() {
        fact = 1;
        for (int i = 1; i <= value; i++) {
            fact *= i;
        }
    }

public:
    Factorial(int n) : value(n), fact(1) {
        calculateFactorial();
    }

    void displayResult() {
        cout << value << "! = " << fact << endl;
    }
};

int main() {
    int n;
    cout << "请输入一个正整数: ";
    cin >> n;
    Factorial f(n);
    f.displayResult();
    return 0;
}

实验 5：Rectangle 与 Point 类（实验5/main.cpp）
------------------------------------------------------------------------------
#include <iostream>
#include <cmath>
using namespace std;

class Point {
public:
    int x;
    int y;
    Point(int xCoord, int yCoord) : x(xCoord), y(yCoord) {}
};

class Rectangle {
public:
    Point bottomLeft;
    Point topRight;

    Rectangle(Point bl, Point tr) : bottomLeft(bl), topRight(tr) {}

    bool isRectangle() {
        return (bottomLeft.x != topRight.x) && (bottomLeft.y != topRight.y);
    }

    int area() {
        if (!isRectangle()) return 0;
        int width = abs(topRight.x - bottomLeft.x);
        int height = abs(topRight.y - bottomLeft.y);
        return width * height;
    }

    int perimeter() {
        if (!isRectangle()) return 0;
        int width = abs(topRight.x - bottomLeft.x);
        int height = abs(topRight.y - bottomLeft.y);
        return 2 * (width + height);
    }

    void display() {
        cout << "左下角: (" << bottomLeft.x << ", " << bottomLeft.y << ")" << endl;
        cout << "右上角: (" << topRight.x << ", " << topRight.y << ")" << endl;
        if (isRectangle()) {
            cout << "面积: " << area() << endl;
            cout << "周长: " << perimeter() << endl;
        } else {
            cout << "无法构成矩形！" << endl;
        }
    }
};

int main() {
    Point p1(0, 0);
    Point p2(5, 3);
    Rectangle rect(p1, p2);
    rect.display();
    return 0;
}

#include <iostream>
using namespace std;

class Complex {
private:
    double real;
    double imag;

public:
    Complex(double r, double i) : real(r), imag(i) {}

    double getReal() const { return real; }
    double getImag() const { return imag; }

    static Complex add(const Complex& c1, const Complex& c2) {
        return Complex(c1.real + c2.real, c1.imag + c2.imag);
    }

    void display() const {
        cout << real;
        if (imag >= 0) cout << " + " << imag << "i";
        else cout << " - " << -imag << "i";
        cout << endl;
    }
};

int main() {
    Complex c1(3.0, 4.0);
    Complex c2(1.5, -2.5);
    cout << "c1 = "; c1.display();
    cout << "c2 = "; c2.display();
    Complex c3 = Complex::add(c1, c2);
    cout << "c1 + c2 = "; c3.display();
    return 0;
}

    double balance;

    IBank(double bal) : balance(bal) {}
    friend double total(const CBank& cBank, const IBank& iBank,
                        const class ABank& aBank);
};

private:
    double balance;
public:
    ABank(double bal) : balance(bal) {}
    friend double total(const CBank& cBank, const IBank& iBank,
                        const ABank& aBank);
};

    cout << "ABank 余额: 3000.25" << endl;

实验 8：Product 商品类（实验8/main.cpp）

#include <iostream>
#include <vector>
#include <iomanip>
using namespace std;

class Product {
private:
    int id;
    double price;
    double discount;
    static double totalSum;
    static int totalSales;

public:
    Product(int id, double p, double d) : id(id), price(p), discount(d) {}

    void calculateSales() {
        double salesPrice = price * (1 - discount / 100.0);
        totalSum += salesPrice;
        totalSales++;
    }

    static double getAveragePrice() {
        if (totalSales == 0) return 0;
        return totalSum / totalSales;
    }

    static void displaySummary() {
        cout << fixed << setprecision(2);
        cout << "总销售额: " << totalSum << endl;
        cout << "总销售数量: " << totalSales << endl;
        cout << "平均价格: " << getAveragePrice() << endl;
    }

    void display() const {
        cout << "商品ID: " << id << ", 原价: " << price
             << ", 折扣: " << discount << "%" << endl;
    }
};

double Product::totalSum = 0;
int Product::totalSales = 0;

int main() {
    vector<Product> products;
    products.push_back(Product(101, 5, 23.5));
    products.push_back(Product(102, 12, 24.56));
    products.push_back(Product(103, 100, 21.5));
    products.push_back(Product(104, 50, 10.0));
    products.push_back(Product(105, 80, 15.0));

    cout << "所有商品信息:" << endl;
    for (const auto& p : products) p.display();

    for (auto& p : products) p.calculateSales();

    cout << "\n销售汇总:" << endl;
    Product::displaySummary();
    return 0;
}

实验 9：Student 学生成绩类（实验9/main.cpp）

#include <iostream>
#include <vector>
#include <string>
#include <algorithm>
using namespace std;

class Student {
public:
    string id;
    string name;
    int prog, net, db;

    Student(string id, string name, int prog, int net, int db)
        : id(id), name(name), prog(prog), net(net), db(db) {}

    int totalScore() const { return prog + net + db; }
    double averageScore() const { return totalScore() / 3.0; }

    void display() const {
        cout << id << "\t" << name << "\t" << prog << "\t" << net << "\t"
             << db << "\t" << totalScore() << "\t" << averageScore() << endl;
    }
};

class StudentInfo {
private:
    vector<Student> students;
public:
    void addStudent(const Student& s) { students.push_back(s); }

    void displayRanking() {
        vector<Student> sorted = students;
        sort(sorted.begin(), sorted.end(), [](const Student& a, const Student& b) {
            return a.totalScore() > b.totalScore();
        });
        cout << "\n===== 成绩排名（按总分降序）=====" << endl;
        cout << "排名\t学号\t姓名\t程序设计\t网络\t数据库\t总分\t平均分" << endl;
        int rank = 1;
        for (const auto& s : sorted) {
            cout << rank++ << "\t";
            s.display();
        }
    }

    void displayHighScorers(double threshold = 85) {
        cout << "\n===== 平均分高于 " << threshold << " 的学生 =====" << endl;
        cout << "学号\t姓名\t程序设计\t网络\t数据库\t总分\t平均分" << endl;
        bool found = false;
        for (const auto& s : students) {
            if (s.averageScore() > threshold) {
                s.display();
                found = true;
            }
        }
        if (!found) cout << "无" << endl;
    }
};

int main() {
    StudentInfo info;
    info.addStudent(Student("2024001", "张三", 90, 85, 88));
    info.addStudent(Student("2024002", "李四", 78, 92, 80));
    info.addStudent(Student("2024003", "王五", 95, 88, 91));
    info.displayRanking();
    info.displayHighScorers(85);
    return 0;
}

实验 10：Circle 圆类（实验10/main.cpp）
------------------------------------------------------------------------------
#include <iostream>
#include <cmath>
using namespace std;

class Point {
public:
    int x, y;
    Point(int xCoord, int yCoord) : x(xCoord), y(yCoord) {}
};

class Circle {
private:
    float* radius;
    Point center;

public:
    Circle(float r, Point c) : radius(new float(r)), center(c) {}

    Circle(const Circle& other) : radius(new float(*other.radius)),
                                   center(other.center) {}

    ~Circle() { delete radius; }

    float getRadius() const { return *radius; }
    Point getCenter() const { return center; }

    static void determineRelationship(const Circle& c1, const Circle& c2) {
        float dx = (float)(c1.center.x - c2.center.x);
        float dy = (float)(c1.center.y - c2.center.y);
        float distance = sqrt(dx * dx + dy * dy);
        float sumRadii = c1.getRadius() + c2.getRadius();
        float diffRadii = fabs(c1.getRadius() - c2.getRadius());

        cout << "圆心距离: " << distance << endl;
        cout << "半径之和: " << sumRadii << endl;
        cout << "半径之差: " << diffRadii << endl;

        if (distance == 0 && diffRadii == 0)
            cout << "关系: 两圆重合" << endl;
        else if (distance == 0)
            cout << "关系: 同心圆" << endl;
        else if (distance > sumRadii)
            cout << "关系: 相离" << endl;
        else if (distance == sumRadii)
            cout << "关系: 外切" << endl;
        else if (distance > diffRadii && distance < sumRadii)
            cout << "关系: 相交" << endl;
        else if (distance == diffRadii)
            cout << "关系: 内切" << endl;
        else
            cout << "关系: 内含" << endl;
    }

    void display() const {
        cout << "圆心: (" << center.x << ", " << center.y
             << "), 半径: " << *radius << endl;
    }
};

int main() {
    Circle c1(5.0f, Point(0, 0));
    Circle c2(3.0f, Point(4, 0));
    cout << "圆1: "; c1.display();
    cout << "圆2: "; c2.display();
    cout << endl;
    Circle::determineRelationship(c1, c2);
    return 0;
}
